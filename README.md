# ROS-Sensor-Data-Processing

## Environment
Ubuntu 18.04, ROS Melodic

## Sensors
* LiDAR: Outser OS1 128CH (1) 
* Camera: GMSL, 1920*1080 (4)
* GPS: RT2000, RTK (1)

## Output Format
* LiDAR: .pcd
* Camera: .png
* GPS: .csv (Latitude, Longitude, Altitude, Speed, Heading, Angle, Roll, Pitch, etc.)
